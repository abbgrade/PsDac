# Get-DacForeignKey

## SYNOPSIS
Returns the foreign keys of a model.

## SYNTAX

```
Get-DacForeignKey [-Model] <TSqlModel> [[-Name] <String>] [<CommonParameters>]
```

## DESCRIPTION
Provides a foreign key object to access it's definition.
